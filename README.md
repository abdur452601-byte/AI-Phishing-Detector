# 🛡️ AI Phishing Email & URL Detector

An intelligent, dual-interface Machine Learning application designed to detect phishing emails and analyze URLs for malicious traits. 

This project was built to provide a reliable first line of defense against cyber threats, satisfying strict command-line executability requirements while also offering a modern, user-friendly Web GUI.

---

## ✨ Features

* **🧠 AI Email Scanner:** Uses a Multinomial Naive Bayes machine learning model trained on a pre-vectorized dataset of over 5,000 emails to predict spam/phishing attempts with high accuracy.
* **🔗 Heuristic URL Checker:** Analyzes web links for common phishing tactics (e.g., exposed IP addresses, missing HTTPS, suspicious keywords, and spoofing symbols) and assigns a dynamic Risk Score.
* **💻 Command-Line Interface (CLI):** A lightweight, pure-terminal executable for rapid testing and system integration.
* **🌐 Interactive Web UI:** A sleek, visual dashboard powered by Streamlit for everyday users.
